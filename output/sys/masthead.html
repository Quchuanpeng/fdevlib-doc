Masthead

介绍

  • logo和logo右侧搜索区域所在那一行容器，称为masthead
  • 搜索区可以个性化，尽量使用统一标准的搜索区功能。

版本

  • @create Edgar 2011.01.27 1.3版
  • @author Edgar 2011.06.13 1.4版
  • @create Denis 2012.02.04 4.0版

使用说明

  • masthead目前暂时由前端架构组负责维护，需要新建或使用现有masthead，需要向架构组申请模版参数(CMS页面)和外部模版地址(External)。接口人：许丹 旺旺：xudanhust；施丹侠 旺旺：swainet(淘宝)
  • 目前全站masthead标准请移步 中国站公共导航应用规范
  • masthead对应的CSS及JS文件需要产品自己引入：
    CSS文件： 
    <link href="http://style.china.alibaba.com/css/sys/universal/masthead/standard-v4-min.css" rel="stylesheet"/>
    JS文件：
    基于FDev4前端框架的页面： 
    <script src="http://style.china.alibaba.com/js/sys/universal/masthead/standard-v4-min.js"></script>
    基于FDev3前端框架的页面： 
    <script src="http://style.china.alibaba.com/js/sys/head/masthead/standard-v4-min.js"></script>
  • 调用方式及合理位置如下： 
    <head>
    ...
    <link href="http://style.china.alibaba.com/css/sys/universal/masthead/standard-v4-min.css" rel="stylesheet"/>
    ...
</head>
<body>
    <div id="doc">
        <div id="header">
            <!--本行为CMS调用代码-->$!sandBoxRender.bandingTemplate("模版参数key")
            <!--本行为外部应用调用代码(应用接入CMS相关操作及问题请联系：颜清国 旺旺:isylinx)-->$cmsTemplate.setTemplate("模版路径")
        </div>
        ...
        <script src="对应FDev框架版本的JS地址"></script>
    </div>
</body>
  • 老版masthead文档，请移步 老版Masthead文档
