Alibar

介绍

  • 页头窄条通栏，包括用户登录状态和网站导航，称为alibar

版本

  • @create Edgar 2010.12.23 2.2版

使用说明

  • alibar目前暂时由前端架构组负责维护，且只有一套版本
  • 目前全站alibar标准请移步 中国站公共导航应用规范
  • alibar对应的CSS及JS文件需要产品自己引入：
    CSS文件： 
    <link href="http://style.china.alibaba.com/sys/css/universal/alibar/standard-v4-min.css" rel="stylesheet"/>
    JS文件：
    基于FDev4前端框架的页面： 
    <script src="http://style.china.alibaba.com/sys/js/universal/alibar/standard-v4-min.js"></script>
    基于FDev3前端框架的页面： 
    <script src="http://style.china.alibaba.com/sys/js/head/alibar/standard-v4-min.js"></script>
  • 调用方式及合理位置如下： 
    <head>
    ...
    <link href="http://style.china.alibaba.com/sys/css/universal/alibar/standard-v4-min.css" rel="stylesheet"/>
    ...
</head>
<body>
    <div id="doc">
        <!--本行为CMS调用代码-->$!sandBoxRender.bandingTemplate("110350_110350_alibar_standard")
        <!--本行为外部应用调用代码-->$cmsTemplate.setTemplate("alibar/standard.vm")
        ...
        <script src="对应FDev框架版本的JS地址"></script>
    </div>
</body>
  • 老版Alibar文档，请移步 老版Alibar文档

Methods

  • Alibar.refresh : 刷新用户状态
  • Alibar.purchaselistRefresh : 刷新进货单信息

Event

  • delitem.alibar : 用户删除进货单数据时候，在document上分发事件

  • Alibar+Masthead

个性化

  • 目前alibar允许个性化，但只限于:底色、文字颜色、下拉菜单隐藏\显示图标、菜单间隔符的修改
  • 不允许擅自修改交互，菜单项图标，添加/删除菜单项，如有相关需要，必须先知会李龙
  • 个性化样式示例： 必须以standard-v4.css为基础，重写相关样式。即先引入standard-v4.css，然后新建css，重写之。

    引入示例：

    <head>
    ...
    <link href="http://style.china.alibaba.com/sys/css/universal/alibar/standard-v4-min.css" rel="stylesheet"/>
    <link href="http://style.china.alibaba.com/XXXX/myalibar.css" rel="stylesheet"/>
    ...
</head>

    代码示例：

    /**底色、上下边线**/
#alibar-v4{
    background-color:#122D43;
    border-top: solid 1px #122D43;
    border-bottom: solid 1px #122D43;
}
/**用户信息区域链接颜色**/
#alibar-v4 .account-sign-status a:link, #alibar-v4 .account-sign-status a:visited{
    color:#FFF;
}
/**[修改会员名]链接颜色**/
#alibar-v4 .account-sign-status a.modify-nick {
    color: #ABC8D9;
}
/**"您好"颜色**/
#alibar-v4 .account-sign-status{
    color:#ABC8D9;
}
/**右侧可点项链接颜色**/
#alibar-v4 .topnav a:link,
#alibar-v4 .topnav a:visited{
    color:#ABC8D9;
}
/**菜单下拉状态，主菜单背景色**/
#alibar-v4 .extra:hover .nav-title,
#alibar-v4 .nav-hover .nav-title{
    background-color:#0A1A28;
}
/**菜单下拉状态，下拉栏背景色**/
#alibar-v4 .nav-content{
    background-color:#0D2333;
}
/**菜单下拉状态，说明项文字颜色**/
#alibar-v4 .nav-content dt{
    color:#608CAC;
}
/**进货单，统计信息背景色**/
#alibar-v4 .topnav-purchaselist .purchase-info{
    background-color:#163952;
}
/**菜单下拉状态，主边框所有边框颜色**/
#alibar-v4 .extra:hover .nav-title, #alibar-v4 .nav-hover .nav-title{
    border-left: solid 1px #0A1A28;
    border-right: solid 1px #0A1A28;
}
/**菜单隐藏/显示图标**/
#alibar-v4 .nav-arrow{
