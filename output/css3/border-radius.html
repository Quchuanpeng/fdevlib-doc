border-radius

定义：

Name: border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius, border-top-left-radius
值: [ <length> | <percentage> ] [ <length> | <percentage> ]?
初始值: 0
适用于: 所有元素
继承性: no
百分比: 相对 border-box 的相应尺寸
媒体: visual
计算值: 两个绝对的<length>或者百分比
Name: border-radius
值: [ <length> | <percentage> ]{1,4} [ / [ <length> | <percentage> ]{1,4} ]?
初始值: 0
适用于: 所有元素，除了当‘border-collapse’属性为‘collapse’的表格元素
继承性: no
百分比: 相对 border-box 的相应尺寸
媒体: visual
计算值: 查看独立的属性

border-*-radius的两个长度值或者百分比定义了一个1/4椭圆的两个半径，定义了一个外边框的形状。第一个值定义了水平方向上的半径，第二个值定义了垂直方向上的半径(如下图所示)。如果第二个值被省略了，则它默认和第一个值相同。如果两个值都是0，则这个角是直角。水平半径的百分比值是相对于border-box的宽度，垂直半径的百分比是相对于border-box的高度。

border-radius这种缩写形式，设置了所有的 border-*-radius 属性。如果斜杠前后都有值，则斜杠前的值定义水平半径，斜杠后的定义垂直半径。如果没有斜杠，则设置水平半径和垂直半径相等。四个值按顺序分别代表 top-left, top-right, bottom-right, bottom-left 。如果bottom-left被省略，则它等于top-right的值，如果bottom-right的值被省略，则它等于top-left的值，如果top-right的值被省略，则它等于top-left的值。

说明：

firefox 3.6 中，需要通过 -moz- 开头的浏览器自定义属性来获得支持，在 firefox 4 中，开始使用标准的属性和值。

safari中，不支持值为百分比的形式。

opera中，值为百分比的形式的时候，表现的比较怪异，非正常计算出来的结果。
