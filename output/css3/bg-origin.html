background-origin

定义：

Name: background-origin
值: <box> [ , <box> ]*
初始值: padding-box
适用于: 所有元素
继承性: no
百分比: N/A
媒体: visual
计算值: 按照指定值

指定值:

        <box> = border-box | padding-box | content-box

对于被作为一个单独的“盒子”渲染的元素，此属性决定背景位置区域。对于作为多个“盒子”渲染的元素（例如 inline box的多行情况），背景位置区域是由'box-decoration-break' 这个属性决定的。

padding-box:
背景位置是相对包含padding但不包含border的区域来计算的。例如设置背景位置为 '0,0'，就设置了背景位置为包含了padding区域的左上角，'100%,100%'就是右下区域。
border-box:
背景位置相对于包含border的区域来计算。
content-box:
背景位置相对 content 区域来计算。

firefox 3.6 中，需要通过 -moz- 开头的浏览器自定义属性来获得支持。并且值也更改为 border, padding, box.下面有详细例子。在 firefox 4 中，开始使用标准的属性和值。

code:

        <div class="show">
            <div class="border"></div>
            <div class="padding"></div>
            <div class="content"></div>
        </div>
        <style type="text/css">
            .show { display:inline-block; }
            .show div {
                float:left;
                width:100px;
                height:100px;
                padding:10px;
                border:10px dashed #666;
                background:url(/fdevlib/static/img/css3/flower-min.png) no-repeat; 
                margin:8px;
                background-position:0 0;
            }
            .show .border {
                -moz-background-origin: border;
                background-origin: border-box;
            }
            .show .paddinig {
                -moz-background-origin: padding;
                background-origin: padding-box;
            }
            .show .content {
                -moz-background-origin: content;
                background-origin: content-box;
            }
        </style>
