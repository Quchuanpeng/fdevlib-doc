background-clip

定义：

Name: background-clip
值: <box> [ , <box> ]*
初始值: border-box
适用于: 所有元素
继承性: no
百分比: N/A
媒体: visual
计算值: 按照指定值

指定值:

        <box> = border-box | padding-box | content-box

此属性决定绘图区域。

border-box:
背景在包含 border 的区域内绘图。
padding-box:
背景在包含 padding 的区域内绘图。
content-box:
背景仅在 content 区域内绘图。

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+
部分支持		 (√)opera 10.6+ (√)safari 5+ (√)IE 9

说明：

在chrome和safari，content-box 需要通过 -webkit-background-clip:content-box;来获得支持。

在firefox中，content-box模式不支持，其余两种定义方式也和标准不同，需要通过-moz-background-clip:border; -moz-background-clip:padding; 来实现。

code:

        <div class="show">
            <div class="border"></div>
            <div class="padding"></div>
            <div class="content"></div>
        </div>
        <style type="text/css">
            .show {
                display:inline-block;
            }
            .show div {
                float:left;
                width:100px;
                height:100px;
                padding:10px;
                border:10px dashed #666;
                background:url(/fdevlib/static/img/css3/flower-min.png) no-repeat;
                margin:8px;
                background-position:-10px -10px;
            }
            .show .border {
                -moz-background-clip:border;
                background-clip:border-box;
            }
            .show .padding {
                -moz-background-clip:padding;
                background-clip:padding-box;
            }
            .show .content {
                -webkit-background-clip:content-box;
                background-clip:content-box;
            }
        </style>
