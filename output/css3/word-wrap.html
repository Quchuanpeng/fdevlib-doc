word-wrap

定义：

Name: word-wrap
值: normal | break-word
初始值: normal
适用于: 所有元素
继承性: yes
百分比: N/A
媒体: visual
计算值: 按照指定
nromal:
仅在允许中断的位置进行中断。
breadword:
一个不能被切断的文字可能在这一行中实在没有其他可以中断的点的时候，被从中间切断。

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (√)IE

说明：

此属性用来定义在一个不能被切断的字符串长到不时候包裹它的容器宽度的时候，是否应该对这个字符串进行切断处理。

例子：

当前浏览器显示效果：

wearealibabafrontendenginners.

wearealibabafrontendenginners.

chrome效果截图：

code:

<div class="show">
	<p class="para1">wearealibabafrontendenginners.</p>
	<p class="para2">wearealibabafrontendenginners.</p>
</div>
<style type="text/css">
	.show p {
		font-size:18px;
		width:180px;
	}
	.show .para1 {
		word-wrap:normal;
	}
	.show .para2 {
		word-wrap:break-word;
	}
</style>
