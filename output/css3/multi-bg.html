multiple background images

定义：

Name: background-image
值: url(A), url(B), url(C)

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (×)IE 6-8
(√)IE 9

说明：

在CSS3中，我们可以为一个元素指定多层背景。背景层数是由background-image属性中由逗号隔开的值的数量决定的。注意，值"none"也会创建一个背景层。

每一个背景图会根据其他背景属性的值来排列、定位、平铺。从第一个值开始匹配。如果某个属性的值的个数小于背景图层的个数，那么浏览器必须计算得出常用的值来进行补全。

浏览器差异：在背景属性中某属性的值的个数比背景图层的个数多的情况下，chrome,safari,opera 表现一致，firefox,IE9表现一致。具体表现为：chrome 等浏览器会把多出的值按顺序继续从background-image中定义的第一个背景图开始匹配创建新的背景图层，应用属性。而firefox等浏览器则把多余的值直接抛弃。在 W3C目前的文档(working draft)中，定义的是直接抛弃多余的属性值。

code:

        <div class="show">
        </div>
        <style type="text/css">
            .show {
                width:500px;
                height:500px;
                background-image:url(./imgs/flower.png), url(./imgs/water.png), url(./imgs/mountain.png);
                background-position: 200px 200px, 20% 80%, top left, right bottom;
                background-origin: border-box, content-box;
                background-repeat: no-repeat;
            }
        </style>

补充：

上述例子 style 在 chrome,safari,opera 等浏览器中等效为：

        .show {
            width:500px;
            height:500px;
            background-image:url(./imgs/flower.png), url(./imgs/water.png), url(./imgs/mountain.png),
                    url(./imgs/flower.png) ;
            background-position: 200px 200px, 20% 80%, top left, right bottom;
            background-origin: border-box, content-box, border-box, border-box;
            background-repeat: no-repeat, no-repeat, no-repeat, no-repeat;
        }

在 firefox, IE9 中等效为：

        .show {
            width:500px;
            height:500px;
            background-image:url(./imgs/flower.png), url(./imgs/water.png), url(./imgs/mountain.png);
            background-position: 200px 200px, 20% 80%, top left;
            background-origin: border-box, content-box, border-box;
            background-repeat: no-repeat, no-repeat, no-repeat;
        }
