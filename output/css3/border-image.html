border-image

定义：

Name: border
值: <'border-image-source'> || <'border-image-slice'> [ / <'border-image-width'>? [ / <'border-image-outset'> ]? ]? || <'border-image-repeat'>
初始值: 请查看各个独立的属性
适用于: 请查看各个独立的属性
继承性: no
百分比: N/A
媒体: visual
计算值: 请查看各个独立的属性
Name: border-image-source
值: none | <image>
初始值: none
适用于: 除了 border-collapse 为 collapse 的 table 内部元素外的所有元素。
继承性: no
百分比: N/A
媒体: visual
计算值: none’ 或者图片

指定一个图片来替代 border-style 属性，并且作为这个元素的一个附加背景图层。如果此值为None,或者图片不能显示，则使用border-style来显示。

Name: border-image-slice
值: [<number> | <percentage>]{1,4} && fill?
初始值: 100%
适用于: 同 border-image-source
继承性: no
百分比: 相对于背景图片的大小
媒体: visual
计算值: 按规定计算

四个 border-image-slice 代表了4个从图片边缘向内的偏移量，把图片划分成9块。四个角，四个边缘，一个中心。中心部分默认会被省略掉（当成全透明处理）。除非有 fill 关键字，详情请参考border-image drawing process

如果第四个值省略，则它和第二个相同，如果第三个省略，则它和第一个相同，如果第二个省略，如果第二个省略，则和第一个相同。

<percentage>：
相对于图片尺寸的比率。
<number>:
数字代表图片的像素或者矢量坐标。
'fill'
如果有 fill 关键字,则引起中间区块被保留。
Name: border-image-width
值: [ <length> | <percentage> | <number> | auto ]{1,4}
初始值: 1
适用于: 同 border-image-source
继承性: no
百分比: 相对于背景图区域的width,height
媒体: visual
计算值: length 定义绝对值，其他按照规定计算

border image 是绘在叫做 “border image area 的区域内的。这是一个和border box 有关的区域。

border-image-width的四个值定义了4个偏移量把border image area 划分成 9 块。他们分别代表从上、右、下、左向内部的距离。

如果第四个值省略，则它和第二个相同，如果第三个省略，则它和第一个相同，如果第二个省略，如果第二个省略，则和第一个相同。

<percentage>：
相对于border image area 尺寸的比率。
<number>:
数字代表border-width 的倍数。
‘auto’
如果指定为 auto，则border image width 是相应的 image slice 的宽度或者高度。如果图片没有必须的固有尺寸，则用相关的border-width 用来代替。
Name: border-image-outset
值: [ <length> | <number> ]{1,4}
初始值: 0
适用于: 同 border-image-source
继承性: no
百分比: N/A
媒体: visual
计算值: length 定义绝对值，其他按照规定计算

border-image-outset 定义了 border image area 在上、右、下、左方向 超出 border-box 的部分。

不允许有负值。

border box 外的部分 border image 的渲染不会触发滚动。并且这一部分对于鼠标事件是不可见的，也不会代表此元素捕获此类事件。

Name: border-image-repeat
值: [ stretch | repeat | round ]{1,2}
初始值: stretch
适用于: 同 border-image-source
继承性: no
百分比: N/A
媒体: visual
计算值: 按照规定

此属性定义了四个边和中间的部分的图片，是怎么缩放和平铺的。如果第二个关键字省略了，则它和第一个一样。

‘stretch’：
这个图片被拉伸来填充区域。
‘repeat’:
这个图片平铺（重复）来填充区域。
‘round’
这个图片平铺（重复）来填充区域，如果重复次数不是整数的话，图片会被缩放来满足整数次。

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (×)IE 9

说明：

此特性各浏览器目前支持不够理想。

所有浏览器都不支持 border-image-outset

chrome || safari 中不支持 border-image-repeat: round ,不支持 border-image-width

opera中不支持 border-image-width

webkit 核心目前通过 -webkit- 前缀获得支持。firefox 通过 -moz-前缀获得支持。
