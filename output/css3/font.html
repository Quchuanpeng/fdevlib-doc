@font-face

定义：

Name: @font-face
值: <'font-description'>

<font-description>一般是这种形式：

descriptor: value;

descriptor: value;

[...]

descriptor: value;

每个 @font-face 规则为每个字体描述符号指定一个值，隐式的或者显示的指定。没有显示指定出来的值将按照初始值处理。

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (√)IE 6

说明：

由于 IE 和其他浏览器使用的字体文件不一样，所以推荐使用 google 的 font-face API 来解决兼容性问题。http://code.google.com/apis/webfonts/

opera不支持 font-face 定义的字体粗体显示。

例子：

当前浏览器显示效果：

we are alibaba front-end engineers.

we are alibaba front-end engineers.

we are alibaba front-end engineers.

chrome中效果截图：

code:

            <div class="show">
                <p class="para1">we are alibaba front-end engineers.</p>
                <p class="para2">we are alibaba front-end engineers.</p>
                <p class="para3">we are alibaba front-end engineers.</p>

            </div>
            <link href='http://fonts.googleapis.com/css?family=Reenie+Beanie' rel='stylesheet' type='text/css'>
            <style type="text/css">
                .show p {
                    font-size:22px;
                }
                .show .para2 {
                    font-family: 'Reenie Beanie';
                }
                .show .para3 {
                    font-family: 'Reenie Beanie';
                    font-weight:bold;
                }
            </style>
