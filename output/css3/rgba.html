RGBA color values

定义：

Name: color: rgba(R,G,B,A)
值: R : 红色值； 正整数|百分比
G : 绿色值； 正整数|百分比
B : 蓝色值； 正整数|百分比
A : 透明度； 取值 0~1 之间

说明：

RGBA 是 RGB 模式 的一个扩展，可以指定颜色的透明度。

和 RGB 的值不同的是， RGBA 的颜色值不能使用16进制来表示。R,G,B的值可以指定0-255之间的一个整数或者0.0%-100.0% 之间的一个百分比。

例子：

当前浏览器显示效果：

Hello,我是黑色有透明度的。

Hello,我是黑色没透明度的。

我有黑色半透明背景

我有黑色不透明背景

我有黑色不透明背景&我这个容器有透明度

code:

        <div class="show">
            <p class="transparent">Hello,我是黑色有透明度的。</p>
            <p class="opaque">Hello,我是黑色没透明度的。</p>
            <p class="trans-box">我有黑色半透明背景</p>
            <p class="opaque-box">我有黑色不透明背景</p>
            <p class="trans-box-2">我有黑色不透明背景&amp;我这个容器有透明度</p>
        </div>
        <style type="text/css">
            .show .container { background-color:#0078AA; padding:10px; width:500px; }
            .show .transparent { color:rgba(0,0,0,0.5); }
            .show .opaque { color:rgba(0,0,0,1); }
            .show .trans-box { background-color:rgba(0%,0%,0%,0.5); color:#FFF; }
            .show .opaque-box { background-color:rgba(0,0,0,1); color:#FFF; }
            .show .trans-box-2 { background-color:rgba(0,0,0,1); color:#FFF; opacity:0.5; }
        </style>
