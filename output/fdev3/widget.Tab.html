经典的Tab切换效果。中文站最常用的js组件之一。

开始：

如果你想使用Tab组件，请在页面中加入以下片段: 
		ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/fdev-min.js");
		ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/yui/animation-min.js");
		ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/widget/tab/tab-min.js");

使用说明

1、本组件对html结构没有要求。调用只需要按顺序将外层容器内的触点(默认为f-tab-t)和内容(默认为f-tab-b)的className一一对应赋给相应的元素即可。 当然触点和内容都必须包含在唯一的外层容器里面。比如:

2、组件调用方法如下：初始化是需要提供2个参数，第一个参数为容器的id(必填)；第二个参数为配置对象(将覆盖默认的参数设定): 

			var tab = new FD.widget.Tab('tab',{});

3、你可以通过设置config.showType来控制需要切换的内容的display属性。这点在对一些inline元素切换时特别有用。 

				var tab = new widget.Tab('tab',{showType：'inline'});

4、你可以通过设置config.startItem来控制初始状态时显示第几个tab元素，记得从0开始计数。 

				var tab = new widget.Tab('tab',{startItem：3});

5、为了防止鼠标划过的误操作。触点的响应事件有0.2妙的延时效果。

6、你可以通过设置config.isLazy来控制是否使用lazy load形式来加载图片，默认为false。 

				var tab = new widget.Tab('tab',{isLazy：true});

7、你可以通过设置config.callback来写回调函数。 

				var tab = new widget.Tab('tab',{callback：function(){/*你的代码*/}});

配置参数列表

配置参数参数可选值作用说明
isAutoPlaytrue | false是否自动进行切换，默认为true
timeDelayNumber自动切换的时间间隔，默认为3(秒)。
eventTypemouse | click切换事件的触发方式，默认为mouse(鼠标划过即触发)。
showTypeblock | inlinebox的容器显示类型，block:块状元素、inline:行内元素，默认为block。
currentClassString触点当前状态的className。默认为block。
tabTitleClassString触点的className。默认为f-tab-t。
tabBoxClassString需要切换的内容的className。默认为f-tab-b。
startItemNumber初始化时所显示的内容序号。默认为0。
isLazytrue | falsefalse：不使用lazy loading, true：使用lazy loading。 默认false
lazyImgStringlazy load 时需用到的专有属性，默认为"data-lazyimg"
callbackFunction回调函数
