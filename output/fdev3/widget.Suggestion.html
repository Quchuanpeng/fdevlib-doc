suggestion组件

根据用户输入的内容进行自动匹配，方便用户输入

更新日志

  • 2010-12-09 version 1.0 beta

开始：

如果你想使用验证组件，请在页面中加入以下片段: 
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/fdev-min.js");
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/yui/get-min.js" type="text/javascript"); 
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/widget/suggestion/suggestion-min.js");

使用说明

1、本组件对html没有特殊要求，只要是文本类型的input即可，通过id初始化即可

2、默认是从http://suggest.china.alibaba.com/bin/suggest取数据，如果数据源不同，可以在参数里配置 

FD.widget.Suggest('suggestion','http://demo.china.alibaba.com/')

节点配置参数列表

配置参数 参数可选值 作用说明
containerClass String 自定义提示层的 class
myParam String 自定义参数，默认为空，组件只发送文本框里的值
containerWidth String 提示层的宽度，默认情况下，提示层的宽度和input输入框的宽度保持一致，示范取值："200px", "10%"等，必须带单位
resultFormat String result的格式，默认"约%result%条结果"
showCloseBtn String 是否显示关闭按钮
closeBtnText String 关闭按钮上的文字
useShim String 是否需要iframe shim
autoFocus true | false 初始化后，自动激活
submitFormOnClickSelect true | false 鼠标点击完成选择时，是否自动提交表单
varName String 存放接口数据的变量名称
leftOffset Number suggest层left偏移量修正
topOffset Number suggest层top偏移量修正
timerDelay Number 定时器的延时
showNum Number 超过数目后显示滚动条,默认为0，表示不做限制
titleLength Number 匹配标题的字数限制，默认为0，不做限制
