开始：

如果你想使用alitalk组件，请在页面中加入以下片段: 
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/fdev-min.js");
/* 当需要获取在线状态时，需要此js文件 */
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/yui/get-min.js");
ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/widget/alitalk/alitalk-min.js");

使用说明

1、组件调用方法如下：第一个参数为旺旺聊天的承载对象(必填)；第二个参数为配置对象(将覆盖默认的参数设定): 

new FD.widget.Alitalk(HTMLDOMElements,{
    //各状态对应的样式
    cls: { base: 'iconAlitalk',on: 'icon-on',off: 'icon-off',mb: 'icon-mb'},
    //旺旺所在的站点，
    siteID: 'cnalichn',
    //是否请求用户在线状态
    remote: true,
    //实例化过程中是否开启自动登陆
    autoLogin: false,
    beginNum: 0,
    prop: '',
    webWW: null,
    onClickBegin: null,
    onClickEnd: null,
    //弹出阿里旺旺下载页
    getAlitalk: function() {
        window.open('http://alitalk.alibaba.com.cn/','_blank');
    }
});

2、定义旺旺样式 

a.iconAlitalk { padding: 2px 0 2px 20px; background: url(http://img.china.alibaba.com/images/fdevlib/contact-icon.gif) no-repeat; line-height: 14px; color: #001D4C; text-decoration: underline !important; font-family: arial,helvetica,sans-serif; font-size: 12px; font-weight: normal; zoom: 1; _padding-top: 3px; }
a.iconAlitalk:hover { color: #FF7300; }
a.icon-on { background-position: -190px -59px; }
a.icon-off { background-position: -190px -29px; }
a.icon-mb { background-position: -190px 1px; }

配置参数列表

配置参数 参数可选值 作用说明
cls { base: 'iconAlitalk',on: 'icon-on',off: 'icon-off',mb: 'icon-mb'} 各状态对应的样式
siteID 'cnalichn' 'cntaobao' 旺旺所在的站点
remote true | false 是否请求用户在线状态
autoLogin true | false 实例化过程中是否开启自动登陆
beginNum 0 指定服务器返回的数组起始索引
prop String || function 添加点击反馈，function必需return字符串
webWW function 自定义未安装旺旺时的webWW功能
onClickBegin function 点击事件触发前事件
onClickEnd function 点击事件触发后事件
getAlitalk function 自定义未安装旺旺事件
