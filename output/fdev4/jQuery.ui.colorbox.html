ui-colorbox : 选色盒子

介绍

轻量级的色彩选择组件，提供常用的色彩供选择。如需专业的拾色功能请使用"ui-colorpicker"

开始

$.use('ui-colorbox', function(){
    $('...').colorbox();
});

Demo

Try it!

Configs

color : String
Default : '#ffffff'
取色器默认选中的颜色，当update配置为true时，该配置失效，取色器会自动验证输入框中的值并进行同步。
transparent : Bolean
Default : false
是否提供透明选择项，用户点击后在"select"事件回调中将返回"transparent"颜色字符串，由使用者自定义该回调的操作。
popup : Bolean
Default : true
取色器的展现方式是否为"浮出层"形式，false则直接渲染到目标容器中。
update : Bolean
Default : false
是否将取色盒与输入框同步，给输入框赋选中的RGB值，或者将输入框变动的值反应到取色盒。
triggerType : String
Default : "click"
当"popup"为true时，触发组件展示的事件名。

Methods

.colorbox( 'toggle' )
取色器展示/关闭。
.colorbox( 'show' )
展示取色器
.colorbox( 'hide' )
关闭取色器
.colorbox( 'setColor', color )
  • color : String 需要设置的色彩，格式为"#******"
设置取色器当前色彩
.colorbox( 'getColor' )
获取取色器当前颜色，没有选择颜色则返回null

Events

change
  • e : Event "mouseover"事件对象
  • o : Object 
    {
    color  //当前鼠标位置所在的颜色值
}
当鼠标move过程中，颜色值的变化值
select
  • e : Event "click"事件对象
  • o : Object 
    {
    color  //选中的颜色值
}
当鼠标点击选中的颜色值
