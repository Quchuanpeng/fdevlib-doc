ui-droppable : 拖放

介绍

配合draggable插件，允许拖放到任何DOM元素内

版本

  • 倪朱国： 创建----2011.10.25

开始

jQuery(function($){
    $.use("ui-droppable",function(){    // 引入ui-droppable的时候，会自动载入依赖项ui-draggable
        $("#draggable").draggable();
        $("#droppable").droppable({
            drop: function( event, ui ){
                $(this).addClass("ui-state-highlight")
                       .find("p")
                       .html("Dropped!");
            }
        });
    });
});

Configs

disabled : Boolean
Default: false
启用或禁用droppable。
accept : Selector, Function
Default: *
仅允许符合选择器的元素激活拖放的事件。
activeClass : String
Default: false
如果设置值，则当有被允许的可拖动元素(acceptable draggable)拖动时， 样式会被添加到该可拖放元素(droppable)中。
addClasses : Boolean
Default: true
设置为false，可以阻止ui-droppable样式添加至可拖放元素中。
greedy : Boolean
Default: false
是否防止嵌套的droppable事件被传播。
hoverClass : String
Default: false
当有被允许的可拖动元素(accptable draggable)移至可拖放元素(droppable)上(hover)时， 为该可拖放元素元添加指定的样式。
scope : String
Default: 'default'
设置元素只允许具有相同scope值的可拖动元素激活此可拖放元素。
tolerance : String
Default: 'intersect'
指定一个可拖动元素被[放到]一个可拖放元素的模式。
可选值：'fit', 'intersect', 'pointer', 'touch'
  • fit: 可拖动元素完全覆盖可拖放元素
  • intersect: 可拖动元素至少50%覆盖可拖放元素
  • pointer: 鼠标指针重叠在可拖放元素上
  • touch: 只要有任何重叠即发生

Events

create Type: dropcreate
droppable的所有事件回调函数都有两个参数： event和ui，浏览器自有event对象， 和经过封装的ui对象。
  • event : Event
  • ui : Object 
    {
    draggable:  // 表示当前被拖动的元素的jQuery对象
    helper:     // 表示当前被拖动的元素的helper对象，即一个jQuery对象（可能就是本身的一个拷贝）
    position:   // 表示当前拖动元素的相对位置(position:relative) { top, left }
    offset:     // 表示被拖动元素的绝对位置(position:absolute) { top, left }
}
创建droppable元素时被触发。
activate Type: dropactivate
一个被允许的(accepted)可拖动元素被拖动时，触发此事件。
deactivate Type: dropdeactivate
一个被允许的(accepted)可拖动元素停止拖动时，触发此事件。
over Type: dropover
当一个被允许(accepted)的可拖动元素移动并覆盖可拖放元素时激活。
out Type: dropout
当一个被允许(accepted)的可拖动元素移出可拖放元素时激活。
drop Type: drop
当一个被允许(accepted)的可拖动元素移动并覆盖可拖放元素，松开鼠标时激活。

Methods

.droppable("destory")
从元素中移除拖放功能。
.droppable("disable")
禁用元素的拖放功能。
.droppable("enable")
启用元素的拖放功能。
.droppable("option", optionName, [value])
获取或设置元素的参数。
.droppable("option", options)
设置元素的参数。
