ui-flash : 创建flash

介绍

本组件提供了flash操作的基本功能，包括：插入flash，浏览器是否支持flash及版本检测。

版本

  • Denis: 增加了flash扩展的注册方法$.util.flash.regist，提高了灵活性，降低耦合度 ---- 2011.06.14

开始

$.use('ui-flash', function(){
    $('...').flash(options);
});

Statics

(readonly)jQuery.util.flash.available : Bolean
判断浏览器是否支持flash。
(readonly)jQuery.util.flash.activeX : Bolean
判断浏览器是否支持flash。该属性暂时不要使用，用available代替
(readonly)jQuery.util.flash.version : Object
  • original : 原生的flash版本的描述信息
  • array : 数组格式的版本信息
  • string : 字符串格式的版本信息
  • major : 数字格式的版本信息
  • minor : 数字格式的版本信息
  • release : 数字格式的版本信息
提供各种格式的flash版本表示。
jQuery.util.flash.encodeParams : Bolean
Default : true
全局配置，是否对flash的param进行encode
jQuery.util.flash.expressInstall : String
Default : 'expressInstall.swf'
当浏览器flash版本低于要求，并且"jQuery.util.flash.expressInstallIsActive"设置为true时，使用本配置指定的flash安装路径。
jQuery.util.flash.expressInstallIsActive : Bolean
Default : false
是否通过expressInstall来升级flash版本

Configs

module : String
Default : false
使用flash扩展组件，需要先 use 对应的flash扩展组件。
* : *
所有配置将反映到flash的param上。 
$.use('ui-flash', function(){
    $('...').flash({
        swf: '/fdevlib/static/flash/test_flashvars.swf',
        wmode: 'transparent',
        flashvars: {
            name1: 'jQuery',
            name2: 'jQuery.ui.flash',
            name3: 'widget'
        }
    });
});
