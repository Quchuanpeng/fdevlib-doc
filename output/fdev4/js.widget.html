widget : Fdev4组件库

介绍

widget提供几乎所有的Fdev4的扩展功能、组件（不包含在core中扩展的部分）。目前包含五个子目录：

  • effect: 动画效果库；
  • ui: jQueryUI组件库；
  • util: jQuery功能组件库
  • web: 网页功能组件库
  • external: 第三方插件所在目录

widget下的命名空间：

  • jQuery.ui: jQueryUI组件
  • jQuery.util: 全局属性/静态方法
  • FE.util: 网站属性/静态方法
  • FE.ui: 网站UI组件
  • FE.sys: 网站模块

Static Method

jQuery.ui.isOverAxis(x, reference, size)
  • x : number x或者y轴坐标
  • reference : number x或者y轴的起始坐标
  • size : number x或者y轴的范围
用来判断坐标是否在范围之内，这是直线范围，如果需要判断区域范围，请使用isOver(x, y, top, left, height, width)
jQuery.ui.isOver(x, y, top, left, height, width)
  • x : number x轴坐标
  • y : number y轴坐标
  • top : number y轴的起始坐标
  • left : number x轴的起始坐标
  • height : number y轴相对起始坐标的高度
  • width : number x轴相对起始坐标的宽度
用来判断坐标是否在范围之内

jQuery.fn.extend

.scrollParent()
获取对象的可滚动的父级节点, 向上递归直到HTML.
.zIndex( [zIndex] )
  • zIndex : Int z-index属性值
  • return : Int z-index值
读取/设置对象的z-index值
.disableSelection()
取消对象的可选中特性, 防止用户拖动鼠标时出现文本选中的高亮状态.
.enableSelection()
激活对象的可选中特性, 使用户拖动鼠标时出现文本选中的高亮状态.
.bgiframe( [options] )
  • options : options 给shim垫片的自定义属性, 默认为透明
给节点添加shim垫片, 默认只有在IE6下才有效, 如果用户自定义了options, 则可以在非IE6下生效.

Selectors extend

":data" ":data(key)"
查询具有data信息的节点, 可以指定对应的key.
":focusable"
查询可以focus的对象,比如表单元素.
":tabbable"
查询具有tabindex属性的对象

Inherit Method

.destroy()
销毁组件实例
.widget()
获取组件实例关联的对象（同this.element）
.option(key[, value])
获取/设置组件配置项
.enable()
激活组件，去掉组件关联的对象的"ui-state-disabled"样式和组件名称+"-disabled"后缀的样式。
.disable()
禁用组件，给组件关联的对象添加"ui-state-disabled"样式和组件名称+"-disabled"后缀的样式。
