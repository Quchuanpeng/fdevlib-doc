FE.ui.Suggestion : 输入框关键字补全

介绍

用于输入框输入关键字补全

版本

  • Denis: masthead中独立改功能模块

开始

$('input[name=keywords]').bind('focus.suggestion', function(){
    $.use('web-suggestion', function(){
        suggestion = new FE.ui.Suggestion(keywords, {
            appendTo: searchKeywords,
            url: currentCfg.url,
            type: currentCfg.type,
            position: {
                my: 'left top',
                at: 'left bottom',
                offset: '-5 0'
            },
            widthfix: 8,
            onSelected: function(e, ui){
                $('button[type=submit]').click();
            }
        });
    });
});

Configs

appendTo : HTML Element | jQuery
下拉列表容器插入的位置
url : Stirng
关键字提示接口地址
type : String
关键字提示类型：目前有"offer"和"company"两种类型
position : Object
下拉列表容器显示的位置偏移，请参考"ui-position"的配置
widthfix : Number
下拉列表的宽度修正，默认跟输入框宽度一致

Methods

.enable()
激活控件
.disable()
禁用控件
.setOptions( o )
  • o : Object 重置一个或多个配置参数
禁用控件

Events

selected : Function
点击选择关键字以后触发
