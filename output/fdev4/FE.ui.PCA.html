FE.ui.PCA : 省市区三级联动菜单

介绍

该插件提供省市区三级联动菜单，最多支持3级联动

版本

  • 施丹侠：创建
  • 倪朱国：修改----2011.11.4

开始

<!--HTML-->
<!--本组件所需的HTML结构很简单，只需要select标签即可：-->
<select></select>
<select></select>
<select></select>
// JavaScript
// 模块动态载入
jQuery(function($){
    $.add("web-pca-data",{
        requires: ["web-pca"],
        js: ["http://" + $.styleDomain + "/js/lib/fdev-v4/widget/web/pca-data-min.js"]
    });
    $.use('web-pca-data',function(){
        // 组件调用方法如下：初始化时需要提供2个参数，
        // 第一个参数为select的数组DOM对象（必填），
        // 第二个参数为配置对象（将覆盖默认的参数设定）。
        var pca=new FE.ui.PCA([p,c,a],{  
            vals: ['330000','330100','330106'],  //初始值，必须匹配数据级联关系
            //texts: ['浙江省','杭州市','滨江区'], //初始值，值或文本初始值两者存一
            showTitle: true,    //是否显示标题
            title: ['--请选择--','--请选择--','--请选择--'],           
            store: FE.ui.PCA.data,   //级联数据
            onPSelected: function() {    //第一个下拉框选项变动后触发 this指向下拉框本身
            },  
            onCSelected: function() {    //第二个下拉框选项变动后触发 this指向下拉框本身
            },  
            onASelected: function() {    //第三个下拉框选项变动后触发 this指向下拉框本身
            }  
        });  
          
    });
});

Configs

vals : Array
Default : []
下拉框初始值，对象option的value值。 例如：
['330000', '330100', '330106']
texts : Array
Default : []
下拉框初始值，对应option的text值，不可同时设置vals参数。例如:
['浙江省', '杭州市', '滨江区']
showTime : Boolean
Default : true
是否显示下拉框的默认标题项
title : Array
Default : ['--请选择--', '--请选择--', '--请选择--']
默认标题项的文本值
store : Array
Default : null
必须设置，下拉数据来源，格式请参考pca-data.js

Events

onPSelected : Function
第一个下拉框出发change事件时触发此方法，this指针指向slect本身
onCSelected : Function
第二个下拉框出发change事件时触发此方法，this指针指向slect本身
onASelected : Function
第三个下拉框出发change事件时触发此方法，this指针指向slect本身
