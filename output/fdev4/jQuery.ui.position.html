ui-position : 定位

介绍

用于相对定位的通用脚本，可以使任何插件相对于窗口(window),文档(document), 一个特定的元素(element),光标/鼠标(cursor/mouse)定位。

版本

  • 倪朱国: 创建 ---- 2011.10.18

开始

$.use("ui-position",function(){
    $("...").position({
        of: $("#parent"),   // 父元素
        my: "left top",     // 自己的基点
        at: "left top",     // 父元素的基点
        offset: "0 0",      // 两基点之间的偏移量
        using: function(){},
        collision: "none"
    });
});

Configs

of : Selector, Element, jQuery, Event
Default : null, 必须指定！
指定父元素。如果你提供的是selector或是jQuery object，他们的第一个元素将被指定为父元素"；如果你提供的是一个 event object，则会使用pageX和pageY属性。
例如："#top-menu"
my : String
Default : "center center"
两个元素要对齐，必须指定各自的基点。my参数用于设置元素本身基点，他的值为一个字符串，分别指定水平和垂直位置。
  • 水平位置可取的值为:"left","center","right";
  • 垂直位置可取的值为:"top","center","right";
如果你只指定了一个值，如"right",脚本会把它理解为"right center";如"top",则理解为"center top"。（符合CSS规范）
at : String
Default : "center center"
at参数用于设置父元素的基点，使用方法可以查考my参数。
offset : String
Default : "0 0"
设置两个基点之间的偏移量。
例如："10 15",表示水平偏移10，垂直偏移15；如果只指定一个值，如"10"，则被解释为"10 10";
collision : String
Default : "flip flip"
当定位的元素溢出窗口时，将它移动到一个替代的位置。和my、at类似，可以接受一对 水平/垂直的值，例如 "flip","fit","fit flip","fit none"
  • flip 根据方向翻转
  • fit 元素被重定位在合适的位置，无法越出边界
  • none 不做冲突检测
