ui-timer : 计时器

介绍

提供倒计时功能。

注意：本组件最大计数时间必须小于10年。

组件会只能查找容器中的year month day hour minute second的样式并渲染结构和数据。

版本

开始

// 模块动态载入
jQuery(function($){
    $.use('ui-timer',function(){
        // ...
    });
});

Demo

Try it!

Configs

format : String
Default : 'y-M-d hh:mm:ss'
设置时间的显示格式，双字母需要在一位数时补0
from : Date
Default: 取客户端机器的当前时间
起始时间。可以由组件调用方传入更加精确的时间（例如服务器当前时间）
to : Date
Default : new Date()
截止时间
step : Integer
Default : 1
跳动频率（单位秒）
animate : Boolean
Default : false
是否动画显示（动画显示需要配合样式文件）
autoStart : Boolean
Default : false
是否自动开始

Methods

.timer("start")
开始计时。ps:只有在倒计时停止状态时才能开始。
.timer("stop")
结束计时，将会触发stop事件。

Events

stop
倒计时结束时触发
