util-cookie : 浏览器cookie操作

介绍

  • 提供完整的浏览器cookie操作（读/写）
  • 由于Core中提供的方法"jQuery.util.subCookie()"依赖于jQuery.util.cookie, 所以附带此方法具有读/写操作

版本

  • Denis: 移植自jQuery官方的cookie插件

开始

$.use('util-cookie', function(){
    $.util.cookie();
    $.util.subCookie();
});

Statics

jQuer.util.cookie( key[, value, options] )
  • key : String 需要获取/设置的cookie的key
  • value : String Cookie的值
  • options : Object 目前只提供"raw"配置，{raw:true}
    • raw : Bolean 是否对value进行encode(escape/unescape)处理，默认false。
读/写cookie
jQuery.util.subCookie( key[, value, options] )
  • key : String 需要获取/设置的子cookie的key
  • value : String 子Cookie的值
  • options : Object
    • raw : Bolean 是否对value进行encode(escape/unescape)处理，默认false。
该方法原理是先通过jQuery.util.cookie方法获取到"alicnweb"的值，然后调用"jQuery.param"方法将值转化为json，最后根据key操作json中的key/value。
