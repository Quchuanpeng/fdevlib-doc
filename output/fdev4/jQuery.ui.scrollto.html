ui-scrollTo : 平滑滚动

介绍

这个插件提供各种配置的动画滚动。

谁滚动？

简单的说，全部匹配的元素，例如:

$("div.pane").scrollTo(...);

如果你想要让窗口滚动可以这样：

$.scrollTo(...);    // the plugin will take care of this

滚到哪里去

这个插件提供了很多方式让你轻松的指定目的地:

  • 一个数字
  • 一个字符串 ('44','100px','+=30px',等)
  • 一个DOM元素 (从逻辑上讲，应该为滚动元素的子元素)
  • 一个选择器
  • 一个特定的字符串"max"，滚到最底部
  • 一个百分数 (例如: 50% ，滚到中间)
  • 一个对象 {top:x, left:y}, x 和 y 可以为上面任何方式的数字和字符串

开始

jQuery(function(){
    $.use("ui-scrollto",function(){
        // scrollTo 和 $.animate() 方法类似，有两种调用方式
        $(...).scrollTo( target, duration, settings);
        $(...).scrollTo( target, settings);
        // 如果使用第二种方法，可以在settings里指定duration的值
    });
});

Configs

axis : String
Default : 'xy'
滚动轴，可选的值有 'x', 'y', 'xy', 'yx'
duration : Number | String
Default : 0
动画的时间长度
easing : String
动画效果的名字（或者说动画的函数）
margin : Boolean
Default : false
如果为值为true，则目的地的计算将会加上元素的border和margin值。
offset : Number | Hash
偏移量，可以为数值也可以为包含left和top的hash值，例如 {offset:50}, {offset:{ left:x, top:y }}
over : Number | Hash
加一小部分元素的高度、宽度，例如{over:0.5}, {over:{left:0.2, top:-0.5}}
queue : Boolean
Default : false
将动作分解，加入到队列中，按先后次序执行，比如 {axis:'xy', queue:true} ，先在X轴上移动，完成后再在Y轴上移动。
axis 的值必须为 'xy' 或者 'yx' 才有意义

Events

onAfterFirst : Function
如果 queue 的值为 true，则在第一个轴上完成动画时触发。
onAfter : Function
动画完成后触发。
