ui-mouse : 鼠标行为

介绍

本组件配合其他有鼠标行为的组件，提供鼠标的“点击”，“拖动”，“释放”的行为跟踪。

版本

  • 基于官方1.8.15版本 ---- 2011.08.26

开始

//在组件制作过程中，需要一个Class具备mouse的特性，可以如下方式定义                
$.widget('ui.portlets', $.ui.mouse, {
    options: {
        //...
    }
    //...
});

Demo

Try it!

Configs

cancel : Selector String
Default : ':input,option'
对符合选择器的元素禁用鼠标动作。
distance : Number
Default : 1
鼠标移动距离的捕捉精度，默认为1px以上的移动都捕捉。
delay : Number
Default : 0
对Mouse Down动作的响应延时，默认为无延时。

Extend & Overriden

_mouseStart( event ) : Function
Default : jQuery.noop
Mousedown之后、Mousemove之前触发此方法
_mouseDrag : Function
Default : jQuery.noop
Mousemove时持续触发此方法
_mouseStop : Function
Default : jQuery.noop
Mouseup时触发此方法
_mouseCapture : Function
Default : jQuery.noop
  • return : Bolean 如果返回"false"则中断监视本次鼠标行为。默认返回"true"。
Mousedown时触发此方法，如果此方法返回"false"则中断监视本次鼠标行为，否则请返回"true"。
