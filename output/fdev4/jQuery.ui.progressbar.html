ui-progressbar : 进度条

介绍

这是一个进度条插件，使用此插件，使用者应该能够提供当前的进度。

版本

  • 倪朱国: 创建 ---- 2011.10.19

开始

jQuery(function($){
    $.use("ui-progressbar",function(){
        $("...").progressbar({
            value: 37
        });
    });
});

Demo

Try it!

Configs

disabled : Boolean
Default : false
使用(false)或禁用(true) progressbar，可以在初始化progressbar的时候设置。
value : Number
Default ： 0
已完成的进度, 0~100.

Methods

.progressbar("destroy")
彻底删除进度条。这会使元素回到他的初始状态
.progressbar("disable")
使进度条失效
.progressbar("enable")
使进度条生效
.progressbar("option",optionName,[value])
  • optionName : String 参数名
获得或更改progressbar的参数 
//getter
var optionName = $("...").progressbar("option",optionName);
//setter
$("...").progressbar("option",optionName,value);
.progressbar("option", options)
  • options : Object 参数对象
提供了一个选项obejct,一次设置多个进度条选项。
.progressbar("widget")
返回 .ui-progressbar 元素
.progressbar("value")
获取、设置进度条的进度

Events

create Type: progressbarcreate
  • event :　Event
progressbar 创建完成时触发
change Type: progressbarchange
  • event :　Event
当进度条的值改变时触发
complete Type: progressbarcomplete
  • event :　Event
当进度条的值达到最大值100时触发
