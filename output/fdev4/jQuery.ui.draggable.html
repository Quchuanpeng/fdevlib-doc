ui-draggable : 拖动

介绍

在任何DOM元素启用拖动功能。通过单击鼠标并拖动对象在窗口内的任何地方移动。

版本

  • 倪朱国：创建----2011.10.24

开始

    $.use('ui-draggable', function(){
        $("...").draggable();
    });

Demo

Configs

disable : Boolean
Default: false
启用或禁用draggable
addClasses : Boolean
Default: true
如果设置成false，将在加载时阻止ui-graggable样式的加载。
当有很多对象要加载draggable()插件的情况下，这将对性能有极大的优化。
appendTo : Element, Selector
Default: 'parent'
helper元素追加的哪个元素后面。
axis : String
Default: false
约束拖动的动作只能在X轴或Y轴上执行，可选值：'x','y'。
cancel : Selector
Default: ':input, option'
防止在指定的对象上开始拖动。
connectToSortable : Selector
Default: false
允许可拖动的元素(draggable)被拖放(dropped)到指定的sortable中，使该可拖动元素成为 sortable的一部分。如果使用此选项helper属性必须 设置成clone才能正常工作
containment : Selector, Element, String, Array
Default: false
强制可拖动元素(draggable)只允许在指定元素或区域的范围内移动。
可选值：'parent', 'docment', 'window', [ x1, y1, x2, y3 ]
cursor : String
Default: 'auto'
指定在做拖动动作时，鼠标的CSS样式。
cursorAt : Object
Default: false
开始移动时，鼠标定位在某个位置上（最多两个方向）。
可选值：{ top, left, right, bottom }
delay : Integer
Default: 0
当鼠标点下后，延迟指定时间后才开始激活拖动动作（单位：毫秒）。此选项可以用来防止鼠标误点。
distance : Integer
Default: 1
单鼠标点下后，只有移动指定像素后才开始激活拖动动作。
grid : Array
Default: false
拖动元素时，只能以指定大小的方格进行拖动。
handle : Element, Selector
Default: false
只能在拖动元素内的指定元素开始拖动。
helper : String, Function
Default: 'original'
拖动元素时的显示方式。
可选值： 'original', 'clone', Function（如果是函数，必须返回一个DOM元素）
iframeFix : Boolean, Selector
Default: false
可防止当mouseover事件触发拖动动作时，移动过iframes并捕获到它（内部内容）， 如果设置成true，则屏蔽层会覆盖页面的iframe。如果设置成对应的选择器，则屏蔽层会 覆盖相匹配的iframe。
opacity : Float
Default: false
当元素开始拖动时，改变元素的透明度。
refreshPositions : Boolean
Default: false
如果设置成true，所有移动过程中的坐标都会被记录。（注意：此功能将影响性能）
revert : Boolean, String
Default: false
当元素拖动结束后，元素回到原来的位置。
revertDuration : Integer
Default: 500
当元素拖动结束后，元素回到原来的位置的时间。
scope : String
Default: true
设置元素只允许拖动到具有相同scope值的元素中。
scroll : Boolean
Default: true
如果设置为true，元素拖动至边缘时，父容器将自动滚动。
scrollSnsitivity : Integer
Default: 20
当元素拖动至边缘时，父窗口一次滚动的像素。
scrollSpeed : Integer
Default: 20
当元素拖动至边缘时，父窗口滚动的速度。
snap : Boolean, Selector
Default: false
当设置位true或元素标签时，元素拖动到其他元素边缘时，会自动吸附其他元素。
snapMode : String
Default: 'both'
确定拖动的元素吸附的模式。
可选值： 'inner', 'outer', 'both'
snapTolerance : Integer
Default: 20
确定拖动的元素移动至其他元素多少像素的距离时，发生吸附的动作。
stack : Selector
Default: false
群组设置，使最后拖动的元素在群组的最上面，即最后被拖动的元素的在zIndex值最大。
zIndex : Integer
Default: false
当拖动元素时，该元素的z-index值。

Events

start Type: dragstart
draggable的所有事件回调函数都有两个参数：event和ui，浏览器自有event对象， 和经过封装的ui对象
  • event : Event
  • ui : Object 
    {
    helper:     // 表示被拖动的元素的JQuery对象
    position:   // 表示被拖动元素的相对位置(position:relative) { top, left }
    offset:     // 表示被拖动元素的绝对位置(position:absolute) { top, left }
}
当鼠标开始拖动时，触发此事件。
drag Type: drag
当鼠标拖动时，触发此事件。
stop Type: dragstop
当鼠标松开时，触发此事件。

Methods

.draggable("destory")
从元素中移除拖动功能。
.draggable("disable")
禁用元素的拖动功能。
.draggable("enable")
启用元素的拖动功能。
.draggable("option", optionName, [value])
获取或设置元素的参数。
.draggable("options",options)
设置元素的参数
.draggable("widget")
返回draggable元素
