core

介绍

此文件是reset、common、grid三类文件的merge版本，也是平时开发中最常用的框架级样式文件。一般页面开发都需要引入此文件。其功能如下：

  • 用来重置浏览器样式，统一各大浏览器的默认表现形式，解决部分兼容性问题。详细请参考reset章节
  • 提供全站公共的样式，样式名称以"fd-"为前缀。详细请参考common章节
  • 提供对应的Grid布局样式。详细请参考grid章节
