ui-tabs, ui-tabs-lazyload, ui-tabs-effect : tab切换组件

介绍

tabs切换组件，例外提供懒加载（图片与HTML的懒加载）与动画效果（渐隐渐现、滚动效果）扩展。其中效果扩展中经过简单配置便可实现焦点图与走马灯效果。
1、经典的tabs切换；
2、tabs切换时可以对图片或HTML进行懒加载；
3、tabs切换时可以有渐隐渐现或滚动效果；

版本

  • hongss: tabs第一版 ---- 2011.02.21

开始

//简单的tabs切换
$.use('ui-tabs',function(){
    $('...').tabs(options);
});
//在tabs中实现懒加载功能
$.use('ui-tabs-lazyload',function(){
    $('...').tabs(options);
});
//在切换时加上动画效果
$.use('ui-tabs-effect',function(){
    $('...').tabs(options);
});

Configs - 基础配置

isAutoPlay : boolean
Default : true
是否自动切换，默认为自动切换
timeDelay : Number
Default : 3
自动切换的时间间隔，单位为“秒”
torpid : Number
Default : 200
事件产生后与切换动作之间的时间差，单位为“毫秒”
event : string
Default : 'mouseover|mouseout'
可触发动作的事件，如：click, mouseover|mouseout
currentClass : string
Default : 'current'
承载当前title触点样式的class名
titleSelector : string
Default : '.f-tab-t'
可以表示title触点集的selector
boxSelector : string
Default : '.f-tab-b'
可以表示box集的selector
selected : number
Default : 0
设置初始化时第几个触点为当前状态，默认从0开始。当设置为负数时，初始化时的当前触点将是随机的。

Configs - 懒加载的配置

lazyImg : string
Default : 'data-lazy-src'
用于承载原img标签中的src属性值的自定义属性名
lazyHtml : string
Default : 'fe-lazy-html'
用于存放需懒加载的HTML的textarea的class名，根据此class名存在与否来判断是否执行html懒加载
isPreLoad : boolean
Default : true
是否预加载下一个tab的数据（img or html），默认为预加载

Configs - 动画效果的配置

effect : string
Default : 'none'
动画效果，可选值：none{无}|scroll{滚动}|fade{渐隐渐现}|function(){自定义}
speed : string/number
Default : 'slow'
动画播放速度，可选值：slow|normal|fast|数值(单位：毫秒)
scrollType : string
Default : 'fill'
滚动方式，loop{循环}|fill{空白填充}|break{不填充}，只对滚动效果生效
perItem : number
Default : 1
每次滚动的子元素数量，只对滚动效果生效
easing : string
Default : 'swing'
动画效果，swing|easeInQuad|easeOutQuad
subLength : number
Default : null
滚动时每个子元素的宽（高）度，默认自动计算，只对滚动效果生效
direction : string
Default : 'left'
滚动方向，可选值：left|right|up|down，只对滚动效果生效

Methods

.tabs('prev', n)
  • n : 可选，number，切换到向前第n个tab
向前n个，默认为向前一个
.tabs('next', n)
  • n : 可选，number，切换到向后第n个tab
可选，向后n个，默认为向后一个
.tabs('length')
  • return : 返回tab数
.tabs('idx')
  • return : 返回当前index
.tabs('select', index)
  • index : 必选；允许负数，如果负数从最后一个开始算
选中所传index的tab
.tabs('reset')
重新遍历tab数；在使用JS动态增加tab后调用，则新增部分也能融入到原tabs中生效。
