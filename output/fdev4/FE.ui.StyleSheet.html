FE.ui.StyleSheet : CSS Rule

介绍

提供动态管理页面级别的Style

版本

  • Denis: 创建
  • 倪朱国： 修改表现样式----2011.10.28

开始

jQuery(function($){
    // 模块动态载入
    $.use('web-stylesheet', function(){
        //可以传入一个现有的style HTMLDOMElement节点
        var ss = new FE.ui.StyleSheet([node]);

        //设置样式
        ss.set(selectors, styleObject);
        //取消样式
        ss.unset(selectors[, styleName]);
        //删除style节点
        ss.destroy();

        //属性
        //保存通过set定义的css规则，可以判断规则是否存在
        ss.cssRules
    });
});

Configs

node : HTMLDOMElement
Default: undefined
需要修改的style节点

Statics

cssRules
通过set定义的css规则，可以判断规则是否存在

Methods

destory()
销毁节点
set( selectors, styleObject )
  • selectors : String 应用样式的元素
  • styleObejct : Object 样式 (键值对)。
设置样式
unset( selectors[, styleName] )
  • selectors : String 取消样式的元素
  • styleName : String | Array 样式名称或样式名称数组
取消样式

如果选中器中包含逗号，将分割成单个选择器，分别适用。

如果取消了样式后，规则中没有其他样式，则该规则将被删除。

styleName(样式名称）该为JavaScript中的样式名称，例如应该是fontSize，而不是font-szie。

enable()
使样式规则可用
disable()
使样式规则不可用
