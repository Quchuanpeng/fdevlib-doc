util-json : JSON操作

介绍

  • 提供对JSON的操作，包括正向、逆向的操作：JSON - 字符串、字符串 - JSON
  • 对于浏览器原生就支持的，框架会智能判断是否需要load此js文件（不影响jQuery.use的正常使用）
  • jQuery核心已经提供了 "字符串 - JSON" 的方法"jQuery.parseJSON()"，请同学们优先使用

版本

  • JSON2.js的版本

开始

$.use('util-json', function(){
    JSON.parse();
    JSON.stringify();
});

Statics

JSON.parse( text[, reviver] )
  • text : String 需要转成JSON的字符串必须是标准格式的字符串，"{a:1}" "{'a':1}"都不是标准的，必须"{"a":1}"
  • reviver : Function 自定义key的value
  • return : Object 转化后的Javascript对象
字符串 - JSON
JSON.parse("{\"success\":true,\"data\":1}");
JSON.stringify( value[, replacer][, space] )
  • value : Object 需要转化为字符串的Javascript对象
  • replacer : Function | Array 自定义key的value的替换
  • space : String 格式化字符串的表现，一般为缩进和换行，默认无格式化效果
  • return : String 转化后的字符串
JSON - 字符串 
JSON.stringify({
    success: true,
    data: 1
});
