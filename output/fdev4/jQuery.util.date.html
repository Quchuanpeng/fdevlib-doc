util-date : 日期

介绍

扩展了原生的Date对象，提供了一些常用的方法。

版本

  • 施丹侠：创建
  • 倪朱国：添加----2011.11.2

开始

jQuery(function($){
    $.use("util-date",function(){
        var dt = Date.parseDate("2011-06-02");
        dt.format("yyyy-MM-dd hh:mm:ss");
    });
});

Properties

Date.monthNames : Array
: Array
一个存储月份名称的数组。
alert(Date.monthNames[0]);  // "January"
Date.abbrMonthNames : Array
一个存储月份名称缩写的数组。
alert(Date.abbrMonthNames[0]);  //"Jan"
Date.format ：String
Default : "yyyy-MM-dd"
日期的默认格式化字符串

Methods

format( [format] )
  • format : String 日期格式。若不指定，则以默认格式处理，默认格式存储在Date.format中，默认值为"yyyy-MM-dd,若不指定，则以默认格式处理，默认格式存储在Date.format中，默认值为"yyyy-MM-dd"。
  • return : String 日期字符串
提供格式，把Date转化成指定格式的字符串。
用例: 
var dt = new Date(),
dateString = dt.format("yyyy-MM-dd hh:mm:ss");

Statics

Date.parseDate( s[, format] )
  • s : String 日期字符串
  • format : String 日期格式。若不指定则使用Date.format中的格式。
  • return : Date 日期对象
提供日期格式，把相应的日期字符串转化为Date对象
用例： 
var dt  = Date.parseDate("2011-11-03");
var dt2 = Date.parseDate("20111103","yyyyMMdd");
var dt3 = Date.parseDate("2011-11-03 10:04:12","yyyy-MM-dd hh:mm:ss");

$.util.date.parse( date, s )
  • date : Date | String 若这个参数为String，则这个参数作为日期运算的表达式，也就不用指定第二个参数了。
  • s ：String 运算的字符串
  • return : Date 运算后的日期
解析日期，包括日期的运算
格式说明： 
 Format         Description
 Y              年
 M              月
 D              日
 W              周
 h              小时
 m              分钟
 s              秒
 today          今天
 now            当前日期和时间
用例： 
$.util.date.parse("2011-11-03");
$.util.date.parse("2011-11-03 13:09:43 - 1M");
$.util.date.parse("2011/4/2 - 2.2h");
$.util.date.parse("today - 1D");
$.util.date.parse("now + 1Y");
$.util.date.parse(new Date());
$.util.date.parse(new Date(), "+1W");
