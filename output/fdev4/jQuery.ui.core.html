ui-core：UI功能扩展+基础类模块

此模块整合了jQuery UI的core.js和widget.js，所有以jQuery UI的模式扩展的组件都需要use此模块：

  • 提供了UI组件常用的一些方法，并重写了一些框架的方法来配合UI组件的完美运作
  • 提供了UI组件的继承机制

版本

  • 基于jQuery UI Core 1.8.13 + jQuery UI Widget 1.8.11

开始

$.add('ui-xxxx', {
    required: ['ui-core'], //注册新模块的依赖
    js: [/*...*/]
});

Properties

jQuery.ui.keyCode.* : Number
  • UP : 方向键“上”
  • DOWN : 方向键“下”
  • LEFT : 方向键“左”
  • RIGHT : 方向键“右”
  • ENTER : 回车键
  • ESCAPE : 空格键
  • NUMPAD_ENTER : 小键盘回车键
  • SHIFT : 功能键“Shift”
  • TAB : 功能键“Tab”
提供了按钮keyCode的枚举。
$(selector).keydown(function(e){
    if(e.keyCode === $.ui.keyCode.ENTER){
        console.info('你敲打了回车键！');
    }
});
jQuery.support.* : Bolean
  • minHeight : 浏览器是否支持"min-height"样式属性
  • selectstart : 浏览器是否支持div的onselectstart事件
扩展浏览器特性检测

jQuery.fn.extend

.scrollParent()
获取最近的具有滚动条的父级jQuery DOM元素。
.zIndex([zIndex])
  • zIndex : 需要设置的"z-index"值
获取/设置jQuery对象的"z-index"样式
.disableSelection()
禁用元素的文本选中能力
.enableSelection()
启用元素的文本选中能力（前提是该元素具有文本选中的能力）
.bgiframe([options])
  • options : 自定义垫片的Style以及在ie6下的iframe的src值
    • src : String iframe的src值，默认是'about:blank'
    • * : String 所有可支持的style属性
使当前元素具有垫片的能力。垫片具有在ie6下遮盖浮层下方select的能力，以及在所有浏览器中防止用户拖动选中下方页面内容的特性。

扩展jQuery选择器

:data(name)
查找 $().data(name) 不为 undefined 的元素
$('input:data(alitalk)')
//return  <input data-alitalk="{id:'swainet'}" />
:focusable
查找能够被"focus"的元素，例如表单元素。
$(':focusable', 'form')
:tabbable
查找具有"tabindex"的元素（按"Tab"键会依次focus此类元素），例如表单元素。
$(':tabbable', 'form')

Classes

jQuery.Widget
  • widgetName : String 组件名称
  • widgetEventPrefix : String 组件事件前缀
  • options : Object 组件配置
  • destroy : Function 销毁组件实例
  • widget() : Function 获取组件容器
  • option( key[, value] ) : Function 获取/设置组件配置
  • enable : Function 启用组件功能
  • disable : Function 禁用组件功能
该类提供了默认的常用组件方法，并且允许用户重写其中的私有方法，包括"_create"，"_init"，"_destroy"等。在组件开发过程中还可以使用类的一些私有方法，包括"_trigger"等。详细的组件开发技巧请参考已有组件或咨询比较熟悉的同学。

Methods

jQuery.widget( name, base, prototype )
  • name : String 组件名称，以"ui."为前缀
  • base : Object 组件的实现
  • prototype : Object 指定组件的prototype对象，默认为"jQuery.Widget"
该方法提供了jQueryUI组件的制作方式，以及一套简单的组件开发模式。所有组件默认都扩展自"jQuery.Widget"。
jQuery.ui.isOverAxis( x, reference, size )
  • x : Number 当前的坐标值
  • reference : Number 边界起始坐标值
  • size : Number 边界长度
这个方法主要为"jQuery.ui.isOver"方法服务。实现原理很简单，同学们可以根据爱好使用。
isOverAxis: function(x, reference, size){
    return (x > reference) && (x < (reference + size));
}
jQuery.ui.isOver( y, x, top, left, height, width )
  • y : Number 当前纵坐标
  • x : Number 当前横坐标
  • top : Number 区域top坐标
  • left : Number 区域left坐标
  • height : Number 区域高度
  • width : Number 区域宽度
判断当前坐标是否在指定范围内。
