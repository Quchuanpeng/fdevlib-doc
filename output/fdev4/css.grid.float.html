float.css : 浮动布局

介绍

目前中文站提供了“浮动”和“双飞翼”两套页面布局结构。这两套布局结构遵行栅格化理论，浮动布局利用容器的左浮自动排列的特性，提供24种浮动样式来控制单列容器的宽度（grid-1 ~ grid-24，单位宽度为32px），适合绝大部分的页面布局。

版本

  • 汉军: 提供了初始版本 ---- 2011-02-23
