web-addons : 浏览器插件（active控件，plugin）应用

介绍

提供判断是否安装了阿里旺旺和阿里工具条两个属性

版本

  • 施丹侠：创建
  • 倪朱国：修改----2011.11.03

开始

// 手动注册模块。由于该模块使用率不高，所以没有在框架核心中注册此组件。需要使用的同学在产品线或者项目global文件中注册模块。
jQuery.add("web-addons", {
    js: ["http://style.c.aliimg.com/fdevlib/js/fdev-v4/widget/web/addons-min.js"],
    ver: "1.0"
});
// 模块动态载入
jQuery(function($){
    $.use("web-addons",function(){
        //...
    });
});

Properties

FE.util.hasAlitalk : Boolean
是否安装了旺旺
FE.util.hasAlitool : Boolean
是否安装了阿里工具条
