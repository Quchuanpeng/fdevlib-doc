ui-flash-clipboard : 使用flash完成复制剪贴板功能

介绍

本组件使用flash来完成复制剪贴板功能。兼容绝大多数浏览器。

版本

  • jianping.shenjp: 参照开源的zeroClipBoard改造完成 ---- 2012.04.25

开始

                $.use('ui-flash-clipboard', function(){
                    $('...').flash(options);
                });

Demo

Try it!

Configs

width : Number
Default : 30
flash容器的宽度(px)。
height : Number
Default : 18
flash容器的高度(px)。
flashvars : Object
Default : 
                    {
                        //(flash有效)为了防止IE下跨域问题，统一设置为500，亲！不要随意更改
                        startDelay: 500,
                        //事件钩子
                        eventHandler: 'jQuery.util.flash.triggerHandler'
                        //根据需要设置style样式，默认情况下不需要写！为了防止编码问题，这里必须用encodeURIComponent包裹
                        //style:encodeURIComponent(styleObj)
                    }
flashvars的配置 
                    默认情况下，flash 按钮的文字为"复制",透明背景。
                    
                    flashvars中的style可以根据需要进行自定义设置，可以设置的属性为：
                    //style的名字必须为clipboard
                    'clipboard{
                        text:复制我吧;//按钮的文本
                        fontSize:19;//按钮文字的大小
                        color:#FF0000;//按钮文字的颜色
                        bgColor:#B1DDF8;//默认的背景色
                        bgColor.hover:#E1F3FD;//hover状态的背景色
                        bgColor.active:#E5F1F7;//鼠标按下时的背景色
                        bgAlpha:1;//背景的透明度
                    }'

Methods

.flash( 'setText', text )
  • text : String 需要复制剪贴板的文本
设置需要复制剪贴板的文本
.flash( 'setHandCursor', enable )
  • enable : boolean 是否使得鼠标移到按钮上时，显示手型
设置是否使得鼠标移到按钮上时，显示手型 (默认情况为手型)

Events

complete
  • event : jQuery.event jQuery事件
  • data : Object flash向外传输的数据 
                                数据结构为
                            {
                                swfid: "clipboard_2"//flash的id
                                text: "copy me!"//复制的文本
                                type: "complete"//事件名
                            }
完成复制分发的事件
mouseOver
鼠标移动到flash按钮上分发的事件
mouseOut
鼠标离开flash按钮分发的事件
mouseDown
鼠标按下flash按钮分发的事件
mouseUp
鼠标抬起flash按钮分发的事件
swfReady
flash准备完毕分发的事件，只有在这个事件分发后，才能正常调用flash里面的方法。
