栅格系统理论说明(32px为单位)

一、fdev栅格系统构建理论

1)、概念
Flowline: 流线
Margin: 边距
Gutter: 槽
Column: 列

2)、基准设定
Gutter(g) = 8px (8为计算机的最小单位)
Margin(m) = 1/2 * Gutter = 4px
Column(c) = 4 * Gutter = 32px

3)、公式推导
设Flowline总宽度为w，Flowline本身由n个column组成
w = c * n + g * (n - 1) + 2 * c
w = 40 * n
设Flowline实际可视宽度为W
W = w - 2m = 40 * n - 8
