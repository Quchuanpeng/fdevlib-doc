fdevlib-example: ColumStater（简单柱状图）

1、纯数值，无css

调用代码示意：

						var colm1 = FD.widget.ColumStater.init('colm1',{data:[['很好',100],['较好',200]]});

2、引用了默认的css，并设置了clength:100,ctype:0

调用代码示意：

						var colm2 = FD.widget.ColumStater.init('colm2',{data:[['很好',100],['较好',200]],clength:100});

3、引用了默认的css，并设置了clength:100,ctype:1（与2的唯一区别是ctype为1）

调用代码示意：

						var colm3 = FD.widget.ColumStater.init('colm3',{data:[['很好',100],['较好',200]],clength:100,ctype:1});

4、引用了默认的css，并使用了自定义模板template:[]（数值后面加上了单位）

调用代码示意：

						var colm4 = FD.widget.ColumStater.init('colm4',{data:[['很好',100],['较好',200]],clength:100,template:['
    ','
  • [%title%]
    0票0%
    • ','
']});

5、引用了默认的css，并使用了topclass参数（配合.topest{color:#f00;}）

调用代码示意：

						var colm5 = FD.widget.ColumStater.init('colm5',{data:[['很好',100],['较好',200]],clength:100,topclass:'topest']});
